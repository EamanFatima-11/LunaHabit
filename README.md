🌸 LunaHabit — Habit Tracker App

Bloom into your best self. One habit at a time.

A beautifully designed Flutter habit tracker with streak system, weekly heat map, insights dashboard, and a stunning rose bloom theme — built to help you build consistent daily routines.

✨ Features

✅ Track Daily Habits — Tap to mark habits complete for today
🔥 Streak System — Tracks current streak and personal best
📅 Weekly Dot View — 7-day completion dots per habit
📊 Progress Ring — Daily completion percentage with animated ring
🏆 Streak Leaderboard — Ranks habits by current streak
🗺️ Weekly Heat Map — Color grid showing completion across all habits
💡 Insights Dashboard — Consistency score, best streak, weekly stats
➕ Add Custom Habits — Name, emoji, color, frequency, and notes
🌸 Animated Splash — Rose bloom entrance animation
📱 3-Tab Layout — Today / Progress / Insights

Installation
bash# Clone the repository
git clone https://github.com/EamanFatima-11/LunaHabit.git

# Navigate to project
cd LunaHabit

# Install dependencies
flutter pub get

# Run the app
flutter run

lib/

🎯 How It Works
1. App loads with 6 sample habits
2. Tap any habit → marks today as complete (dot fills)
3. Streak counter increases automatically
4. Progress tab → leaderboard + weekly heat map
5. Insights tab → consistency score + tips
6. FAB button → add your own custom habit

🌸 Habit Frequencies
Daily · Weekdays · Weekends · 3x/week
